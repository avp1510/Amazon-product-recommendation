# Amazon Product Recommendation

This repository contains a notebook-based recommendation system project focused on Amazon product suggestions using collaborative filtering.

## Repository Contents

- `amaz_rec.ipynb`: primary notebook for experimentation, modeling, and recommendation generation

## Skills Demonstrated

- Recommender systems
- Collaborative filtering
- Notebook-based machine learning experimentation
- E-commerce personalization workflows

## Notes

The project is currently organized as a single exploratory notebook rather than a packaged application.
